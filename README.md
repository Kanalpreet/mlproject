# 🎓 Student Performance Prediction System

An end-to-end machine learning system that predicts student performance based on demographic and academic features. Built using a production-level pipeline and deployed as a web application.

---

## 🚀 Overview

This project demonstrates how real-world ML systems are designed — covering the complete pipeline from data ingestion to deployment.

The system predicts a student's math score using multiple input features and provides real-time predictions through a Flask-based web interface.

---

## 🎯 Key Features

- End-to-end ML pipeline (data → preprocessing → training → deployment)
- Automated data ingestion and transformation
- Model training with hyperparameter tuning (GridSearchCV)
- High-performance model using CatBoost
- Modular, scalable project architecture
- Real-time prediction via web application

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, CatBoost  
- **ML Concepts:** Regression, Feature Engineering, Model Evaluation  
- **Backend:** Flask  
- **Deployment:** Render  

---

## ⚙️ Pipeline Architecture

1. Data Ingestion  
2. Data Transformation  
3. Model Training  
4. Model Evaluation  
5. Deployment  

---

## 📊 Model Performance

- Achieved **~85–90% prediction accuracy**
- Optimized using **GridSearchCV**
- Selected CatBoost for better performance on tabular data
