Student Performance Prediction

An End-to-End Machine Learning application that predicts student math performance using demographic and academic indicators. Built with production-grade ML pipeline & deployment architecture.

🔮 Start Prediction

Why This Project?

⚙ End-to-End Pipeline

Data Ingestion → Data Transformation → Model Training → Deployment with reusable ML architecture.

🤖 Advanced ML Models

Uses XGBoost, CatBoost, Random Forest & Linear Models with hyperparameter tuning.

🌍 Production Deployment

Fully deployed on cloud with Flask backend, real-time predictions & scalable design.

How It Works

1️⃣ Input Student Details

Enter gender, ethnicity, parental education, lunch type, and test preparation details.

2️⃣ ML Processing

Data is preprocessed using a trained ColumnTransformer and fed into the best-performing regression model.

3️⃣ Instant Prediction

Get predicted math performance score instantly.
